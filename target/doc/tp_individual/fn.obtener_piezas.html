Expand description

Recibe las líneas del archivo (de tipo Lines<BufReader<File>>) y devuelve las piezas que se encuentran en el mismo. Casos de retorno:

  • (None, None): No se encontraron piezas en el archivo
  • (Some(pieza), None): Solo se encontró una blanca en el archivo
  • (None, Some(pieza)): Solo se encontró una negra en el archivo
  • (Some(pieza1), Some(pieza2)): Se encontraron ambas piezas en el archivo (blanca y negra) De esta manera, se podrá luego validar si las fichas son válidas o no. Verificacion de las dimensiones del tablero:
  • Para las columnas se leerán como máximo 8, si hay más, se descartan
  • De existir más de 8 filas, se devolverá un error En caso de algun error devolverá el mismo y lo mostrará por pantalla.
